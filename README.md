## Technologies Used
* React.js
* Node.js
* Express
* Cors
* Axios
* And other libraries

## Prerequisites
* Installed Node.js https://nodejs.org/en/download/

## Running the App
* Install all the packages in the root directory
  ```
    npm install
  ```
* Running the website in http://localhost:3000/ and server in http://localhost:5000/
  ```
    npm start
  ```
